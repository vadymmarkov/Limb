## Installation

`Limb` is available through [Swift Package Manager](https://github.com/apple/swift-package-manager).
To install it, simply add the following lines to your `Package.swift`:

```swift
.Package(url: "https://github.com/vadymmarkov/Limb.git", versions: Version(0,1,0)..<Version(1,0,0))
```
